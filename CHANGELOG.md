# CHANGELOG — 논문 작성 서포터

## [0.2.0] — 2026-07-11 · Sprint 2 프로젝트 관리 확장 (SPEC-TSA-002)

Sprint 1 실기기 피드백 반영. 앱 전체가 단일 연구가 아닌 **여러 연구 프로젝트**를 병행 관리하는 구조로 확장되었고, 5탭(대화/리서치/보관함/글쓰기/설정) 구성으로 재편되었다.

### 추가
- 프로젝트 전환: 상단에서 여러 연구 프로젝트를 만들고 한 번의 클릭으로 전환. 전환 시 메모리·보관함·대화·리서치 이력·점검 기록이 선택한 프로젝트 기준으로 즉시 바뀌며 이전 프로젝트 화면이 남지 않음. 기존 Sprint 1 데이터는 최초 실행 시 "내 연구 1"로 자동 편입
- 문헌 보관함: 리서치 결과의 참고문헌·관련문헌에 저장 버튼(체크) 추가 → 프로젝트별 보관함에 저장, 목록 조회·한 줄 메모·삭제·원문 링크 열기 지원
- 서지 복사(APA): 보관함에서 문헌을 선택해 APA 형식으로 클립보드 복사(국문/영문 표기 규칙 각각 적용)
- 리서치 이력: 딥리서치 완료 시 질문·리포트·참고문헌을 자동 저장, 이력 목록에서 과거 결과 재열람(재검색 불필요)
- 회의하기 연계: 리서치 결과(또는 이력)에서 "이 결과로 회의하기"를 누르면 리포트 요약과 참고문헌 목록이 담긴 새 대화가 채팅 탭에서 바로 시작됨
- 대화 기록 저장/이어하기: 아이디어 회의 대화가 매 턴 자동 저장되며, 대화 목록(제목·날짜)에서 선택해 이어가거나 새 대화를 시작할 수 있음
- 글쓰기 탭 4분할: 서론 점검·문장 다듬기(국문/영문 학술 문체 교정)·모의 심사(Reviewer 2 관점 예상 질문)·점검 기록(과거 결과 재열람)을 한 탭에서 제공
- 자동 백업: 세션마다 data 폴더 전체를 zip으로 백업(최근 5개 보관)
- 딥리서치 체크포인트 이어하기: 진행 중 중단되어도 재실행 시 마지막 진행 지점부터 재개

### 보안
- 레코드 id(리서치 이력·대화·점검 기록·모의 심사) 경로 탈출 방지 가드를 핸들러·저장소 양쪽에 이중 적용
- 리서치/품질 게이트 실행 중 프로젝트 전환이 발생해도 실행 중이던 프로젝트 경로로만 기록되도록 진입 시점 경로 고정
- 마이그레이션 실패 시 롤백, 백업 경로의 특수문자(아포스트로피 등) 처리 보강

### 검증
- 테스트 507→832개(Sprint 2 신규 325개), 전 구간 typecheck·build 통과, 종합 E2E 6개 시나리오(프로젝트 전환/데이터 격리, 리서치→이력→회의하기, 대화 CRUD, 글쓰기 관통, 체크포인트 재개) 통과
- Phase 4 리뷰·보안 감사 발견 High 2건 스프린트 내 수정 완료
- 이연(백로그): 예외/결과값 반환 규약 통일, 저장 로직 공통화(M3/M4) — 다음 스프린트

## [0.1.0] — 2026-07-11 · Sprint 1 MVP (SPEC-TSA-001)

첫 동작 버전. Electron 포터블 앱으로서 설정 마법사 → 채팅(아이디어 회의·논문 찾기) → 서론 점검까지의 여정이 동작한다.

### 추가
- 연구 프로젝트 메모리 코어(JSON): 연구 질문·가설·용어·결정 이력·지도교수 피드백, 모든 LLM 호출에 자동 주입
- 선행연구 딥리서치: 국문+영문 검색어 생성 → KCI·ScienceON·Semantic Scholar 병렬 조회 → 관련도 스크리닝 → 실링크 리포트(서지정보는 API 응답에서만 결정론적 조립)
- 채팅 아이디어 회의: 결정 제안 태그 → 사용자 확인 후 연구 이력 기록, 대화 자동 컴팩션
- 서론 품질 게이트: 연구 갭·기여(LLM)·인용 존재(규칙) 3기준, 2단계 경고식 완료 차단, "✍️ 서론 점검" 탭
- 멀티 LLM: Claude/Gemini/OpenAI 어댑터(SDK 의존성 0), 무료 모드 rate limiter(RPM 8), 에러 한국어 번역 + 선별 자동 재시도
- 최초 실행 마법사: 무료/유료 모드 안내(품질·프라이버시), 키 연결 확인 후 DPAPI 암호화 저장
- 포터블 안전장치: zip 미해제/임시폴더 실행 감지, 처음 사용자 안내 HTML(SmartScreen 대처), app/data/config 3분리
- 원격 설정 이중화: settings.json + GitHub Pages endpoints.json(실패 시 알림 후 로컬 폴백)
- 학술 API 키 3단 폴백: 사용자 키 > 내장 공용 키 > mock
- 배포 zip 조립 스크립트(scripts/package-zip.mjs)

### 보안
- 원격 endpoints 병합에 https + 서비스별 호스트 화이트리스트(API 키 유출 경로 차단)
- 전 IPC 핸들러 런타임 payload 검증, 렌더러 CSP, 외부 링크 허용 목록
- 키는 연결 확인 성공 후에만 저장(미검증 키 잔존 버그 수정)

### 검증
- 테스트 297개(단위+모듈 통합 E2E), 도메인 로직 커버리지 94~99%
- 이연: GUI/패키징 실기기 E2E, 체크포인트 재개, 키 제외 내보내기 → Sprint 2
