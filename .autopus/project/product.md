# product.md — 논문 작성 서포터

> `/auto sync` 갱신. 마지막 갱신: 2026-07-11 (Sprint 2, SPEC-TSA-002)

## 제품

한국 대학원생(컴퓨터 비숙련자)용 논문 작성 보조 데스크톱 앱. 설치 없는 포터블 zip으로 배포되며, 연구 프로젝트 메모리 하나가 모든 AI 기능의 공유 컨텍스트가 되는 것이 차별점. Sprint 2부터는 **여러 연구 프로젝트를 동시에 병행 관리**할 수 있다 — 모든 데이터(메모리·보관함·대화·리서치 이력·점검 기록)가 프로젝트 단위로 완전히 격리된다.

## 화면 구성 (5탭)

| 탭 | 내용 |
|---|---|
| 💬 대화 | 아이디어 회의 채팅(자유 대화 + 딥리서치 모드 토글), 대화 목록에서 이어하기/새 대화 |
| 🔍 리서치 | 딥리서치 실행·결과, 리서치 이력 목록(재열람), "이 결과로 회의하기" |
| 📚 보관함 | 저장한 문헌 목록, 한 줄 메모, 삭제, 원문 링크, APA 서지 복사 |
| ✍️ 글쓰기 | 서론 품질 게이트·문장 다듬기·모의 심사·점검 기록 4분할 |
| ⚙️ 설정 | 마법사로 구성한 설정 재확인/변경 |

상단의 프로젝트 전환 UI(ProjectSwitcher)는 5탭 모두에 공통으로 걸쳐 있으며, 전환 시 직전 탭을 유지한 채 데이터만 교체된다.

## 핵심 기능

| 기능 | 설명 | 도입 |
|---|---|---|
| 설정 마법사 | 무료(Gemini)/유료 모드 선택, 품질·프라이버시 안내, 키 연결 확인 후 암호화 저장 | Sprint 1 |
| 프로젝트 관리 | 여러 연구 프로젝트 생성·전환·이름변경·보관(soft-delete), 데이터 완전 격리, Sprint 1 데이터 자동 마이그레이션 | Sprint 2 |
| 아이디어 회의 채팅 | 연구 메모리 자동 주입 상담, 대화 중 결정을 확인 후 연구 이력으로 기록, 이력 자동 컴팩션, 매 턴 자동 저장+이어하기 | Sprint 1 (영속화 Sprint 2) |
| 선행연구 딥리서치 | 국문+영문 검색어 생성 → KCI·ScienceON·Semantic Scholar 병렬 조회 → 스크리닝 → 실링크 리포트, 진행 체크포인트 이어하기 | Sprint 1 (체크포인트 Sprint 2) |
| 문헌 보관함 | 참고문헌/관련문헌 저장 버튼(체크) → 프로젝트별 보관, 메모·삭제·링크, APA 서지 복사 | Sprint 2 |
| 리서치 이력 | 딥리서치 완료 시 자동 저장, 이력 목록에서 재열람·재활용 | Sprint 2 |
| 회의하기 연계 | 리서치 결과(또는 이력)에서 요약+참고문헌을 새 대화로 핸드오프(토큰 예산 캡 적용) | Sprint 2 |
| 서론 품질 게이트 | 연구 갭·기여 명시(LLM)·인용 존재(규칙) 3기준 검사, 2단계 경고식 완료 차단, 점검 기록 저장/재열람 | Sprint 1 (기록 Sprint 2) |
| 문장 다듬기 | 국문/영문 문단을 학술 문체로 교정, 변경 사유 제시 | Sprint 2 |
| 모의 심사 | Reviewer 2 관점 예상 질문·약점(단일 모델 역할극), 결과 저장 | Sprint 2 |
| 자동 백업 | 세션 시작 시 data 폴더 전체 zip 백업, 최근 5개 보관 | Sprint 1 설계 (실행 Sprint 2) |
| 안전장치 | zip 미해제 실행 감지, 한국어 에러 번역+자동 재시도, 무료 모드 rate limiter, 원격 설정 이중화 | Sprint 1 |

- 학술 검색: 사용자 키 > 내장 공용 키 > mock 폴백. Semantic Scholar는 키 없이 실동작.
