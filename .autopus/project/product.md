# product.md — 논문 작성 서포터

> `/auto sync` 갱신. 마지막 갱신: 2026-07-11

## 제품

한국 대학원생(컴퓨터 비숙련자)용 논문 작성 보조 데스크톱 앱. 설치 없는 포터블 zip으로 배포되며, 연구 프로젝트 메모리 하나가 모든 AI 기능의 공유 컨텍스트가 되는 것이 차별점.

## 핵심 기능 (Sprint 1 구현 완료)

| 기능 | 설명 |
|---|---|
| 설정 마법사 | 무료(Gemini)/유료 모드 선택, 품질·프라이버시 안내, 키 연결 확인 후 암호화 저장 |
| 아이디어 회의 채팅 | 연구 메모리 자동 주입 상담, 대화 중 결정을 확인 후 연구 이력으로 기록, 이력 자동 컴팩션 |
| 선행연구 딥리서치 | 국문+영문 검색어 생성 → KCI·ScienceON·Semantic Scholar 병렬 조회 → 스크리닝 → 실링크 리포트 (전형 LLM 3회 호출) |
| 서론 품질 게이트 | 연구 갭·기여 명시(LLM)·인용 존재(규칙) 3기준 검사, 2단계 경고식 완료 차단 |
| 안전장치 | zip 미해제 실행 감지, 한국어 에러 번역+자동 재시도, 무료 모드 rate limiter, 원격 설정 이중화 |

## 사용자·모드

- 무료 모드: Gemini Flash 무료 티어 (RPM 8 보수 제한). 원고 업로드 비권장 안내(학습 사용 가능성).
- 유료 모드: Claude/Gemini/OpenAI 키 등록.
- 학술 검색: 사용자 키 > 내장 공용 키 > mock 폴백. Semantic Scholar는 키 없이 실동작.
